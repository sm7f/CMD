# Devolução
Para emitir uma nota fiscal de devolução, é necessário ter os seguintes parâmetros cadastrados no sistema:

1. Produtos cadastrados no sistema: Certifique-se de que os produtos envolvidos na devolução estão devidamente registrados no sistema.

2. Fornecedor cadastrado no sistema: O fornecedor relacionado à nota original deve estar corretamente cadastrado no sistema.

3. Chave da nota original (NF-e): A chave de acesso da nota fiscal de origem é fundamental para referenciar a devolução à operação inicial. Certifique-se de ter a nota em mãos ou uma cópia (PDF) disponível para consulta.

# Reparo e Conserto

Para emitir uma nota fiscal de Reparo e Conserto, é necessário ter os seguintes parâmetros cadastrados no sistema:

1. Produtos cadastrados no sistema: Certifique-se de que os produtos enviados para reparo estão devidamente registrados.

2. Destinatário cadastrado no sistema: O fornecedor ou assistência técnica responsável pelo conserto deve estar corretamente cadastrado.

3. CFOP adequado: O resposavel que está solicitandoa nota precisa te informar o CFOP que será utilizado. 

4. Nota Fiscal Original (se aplicável): Caso o produto tenha sido adquirido anteriormente, informe a chave da NF-e original.
